# /fix-bug — Исправить баги

## Описание
Прочитать список открытых багов из `.claude/memory/known-bugs.md` и исправить их в коде.

## Шаги

1. Прочитай `.claude/memory/known-bugs.md`
2. Найди все баги со статусом `открыт`
3. Для каждого открытого бага:
   a. Прочитай указанный файл и строку из поля **Где**
   b. Пойми суть проблемы (поля **Воспроизведение**, **Ожидание**, **Реальность**)
   c. Исправь баг в коде
   d. Обнови статус бага в `known-bugs.md` на `исправлен` и добавь дату исправления
4. Если открытых багов нет — сообщи об этом

## Правила

- Не трогай баги со статусом `исправлен` или `отложен`
- Соблюдай `svelte-style-guide.md` при исправлениях
- Не рефакторь код за пределами проблемного места
- После всех исправлений выведи краткую сводку: что исправлено
