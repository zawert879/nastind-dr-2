# /implement-component — Реализовать задачу

## Описание
Взять задачу из dev-plan.md и реализовать код.

## Вход
Название задачи или описание того, что нужно сделать.

## Шаги

1. Прочитай `.claude/memory/dev-plan.md` — найди задачу
2. Прочитай `.claude/rules/svelte-style-guide.md` — стиль кода
3. Прочитай `.claude/memory/architecture-decisions.md` — ограничения
4. Переключись в режим Web Engineer (`.claude/agents/web-engineer.md`)
5. Реализуй задачу:
   - Создай / измени файлы
   - Следуй шаблонам из `.claude/templates/`
   - Убедись что код компилируется
6. Обнови `dev-plan.md` — отметь задачу выполненной

## Ограничения

- Не создавать `+page.server.ts`, `+server.ts`
- API-вызовы только через `src/lib/api/`
- TypeScript strict, без `any`
- Один компонент = один файл

## Чеклист перед завершением

- [ ] Код компилируется
- [ ] Типы корректны
- [ ] Нет серверных файлов
- [ ] Стиль соответствует гайду
- [ ] dev-plan.md обновлён
