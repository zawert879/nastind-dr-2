# Known Bugs — Трекер багов

> Известные баги. Агент обновляет при обнаружении, закрывает при исправлении.

## Формат записи


### BUG-001 — gray-matter несовместима с браузером (Buffer is not defined)

- **Статус**: исправлен
- **Дата исправления**: 2026-04-03
- **Решение**: заменена на `front-matter@4.0.2` (browser-compatible). API: `fm(raw)` → `{ attributes, body }` вместо `matter(raw)` → `{ data, content }`

---

### BUG-002 — Отсутствуют изображения-заглушки

- **Статус**: исправлен
- **Дата исправления**: 2026-04-03
- **Решение**: созданы SVG-заглушки в `static/images/covers/` (3 шт.) и `static/images/avatars/` (3 шт.). Frontmatter статей обновлён с `.jpg` на `.svg`.

---

### BUG-004 — Контент статьи не обновляется при навигации через "Похожие статьи"

- **Статус**: исправлен
- **Дата исправления**: 2026-04-03
- **Приоритет**: критический
- **Где**: `src/routes/articles/[slug]/+page.svelte`
- **Причина**: `onMount` срабатывает один раз при монтировании компонента. SvelteKit переиспользует компонент при клиентской навигации между `[slug]`-роутами — `onMount` не перезапускается.
- **Решение**: заменён `onMount` на `$effect` — реактивно отслеживает изменение `$page.params.slug` и перезагружает данные при каждом переходе.

---

### BUG-003 — Нет мобильной навигации

- **Статус**: исправлен
- **Дата исправления**: 2026-04-03
- **Приоритет**: высокий
- **Где**: `src/routes/+layout.svelte:27` — `nav.hidden.md:flex`
- **Воспроизведение**: открыть сайт на экране < 768px
- **Ожидание**: доступна навигация (бургер-меню или горизонтальный скролл)
- **Реальность**: навигационные ссылки полностью скрыты, альтернативы нет
- **Дата**: 2026-04-02
- **Контекст**: PRD прямо указывает "просмотр с мобильного" как приоритет аудитории. Исправить в Фазе 2 при создании полноценной навигации.
- **Решение**: добавлен бургер-кнопка (`md:hidden`) с `$state(menuOpen)` (Svelte 5 runes). При нажатии — полноширокий dropdown с теми же навигационными ссылками; закрывается при клике на ссылку.
